음성인식을 활용한 회의록 자동 문서화 어플리케이션

Downlod APK Source Code

펜과 노트가 필요없어요

앱을 사용해서 회의를 진행한다면 모든 데이터가 서버에 저장되며, 나중에 다시 확인할 수 있습니다.

자동으로 문서로 바꿔줍니다

다운로드 버튼만 누르면 스마트폰에 회의록 파일이 생성됩니다.

요약하는데 시간을 많이 쏟지 마세요

회의록을 손쉽게 요약할 수 있는 템플릿을 제공합니다.

회의를 체계적으로 진행해요

회의 중에는 동시에 발언할 수 없으며, 사회자 주도 하에 진행합니다.

계정 연동

페이스북 또는 카카오 계정을 사용합니다.

회의방 생성 및 회의자 초대

방 이름만 입력하고 회의자들의 아이디를 사용해서 인원을 추가합니다.

실시간

실시간으로 알림이 도착하고 실시간으로 대화가 표시됩니다.

자동 문서화

회의한 내용을 PDF 또는 Word 파일로 다운로드 할 수 있습니다.

단어 수정

음성인식 결과 잘못 인식된 문장은 바로바로 수정할 수 있습니다.

단어 통계 및 요약

몇가지 유용한 부가 기능들이 있습니다.

동영상 보러 가기

시작 화면

시작 화면에서는 로그인 절차가 진행됩니다.

전화번호

아이디로 사용됩니다

카카오 계정 로그인
페이스북 계정 로그인

회의 진행 화면

실시간으로 회의가 진행되는 화면입니다.

음성인식

발언하기 버튼을 누르면 안드로이드 내장 음성인식 API를 활용하여 텍스트로 변환해줍니다.

FCM

음성인식 결과를 서버로 전송하면 FCM을 활용해서 실시간으로 모든 회의자에게 전송됩니다.

회의록 자세히보기 화면

발언 수정하기

발언을 선택하여 수정할 수 있습니다.

PDF 또는 Word 파일로 다운로드하기

서버에서 회의록의 모든 내용을 PDF 또는 Word 파일로 변환하여 전송해줍니다.

기타 부가 기능들

단어 통계 보기, 회의 내용 요약하기
